                        API 키는 어디에 저장되나요?
                    
    
                    
    
                        입력하신 API 키는 브라우저의 로컬 스토리지에만 저장됩니다.
                        이는 쿠키처럼 여러분의 컴퓨터에만 저장되는 정보로, 외부 서버로 전송되지 않습니다.
                    
    
                
    

                
    
                    
    
                        🔗
                        어떻게 작동하나요?
                    
    
                    
    
                        이 웹사이트는 여러분의 브라우저에서 직접 OpenAI API에 연결됩니다.
                        중간 서버를 거치지 않으므로 제3자가 여러분의 API 키나 생성된 내용을 볼 수 없습니다.
