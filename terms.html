이용약관

시행일: 2026년 1월 1일 | 최종 수정: 2026년 1월 27일

제1조 (목적)

이 약관은 AI 블로그 도구(이하 "서비스")가 제공하는 서비스의 이용조건 및 절차, 이용자와 서비스 간의 권리, 의무 및 책임사항을 규정함을 목적으로 합니다.

제2조 (정의)

  1. "서비스"란 AI 블로그 도구가 제공하는 블로그 콘텐츠 생성, SEO 분석, 유사도 검사 등의 온라인 서비스를 말합니다.
  2. "이용자"란 본 약관에 따라 서비스가 제공하는 서비스를 이용하는 자를 말합니다.
  3. "API 키"란 이용자가 AI 서비스(OpenAI, Google Gemini, Anthropic Claude 등)를 이용하기 위해 직접 발급받아 입력하는 인증 키를 말합니다.

제3조 (약관의 효력 및 변경)

  1. 본 약관은 서비스 화면에 게시하거나 기타의 방법으로 이용자에게 공지함으로써 효력이 발생합니다.
  2. 서비스는 합리적인 사유가 발생할 경우 관련 법령에 위배되지 않는 범위에서 본 약관을 변경할 수 있습니다.
  3. 약관이 변경되는 경우 서비스는 변경 사항을 시행일 7일 전부터 공지합니다.
  4. 이용자가 변경된 약관에 동의하지 않는 경우 서비스 이용을 중단할 수 있습니다.

제4조 (서비스의 제공)

  1. 서비스는 다음과 같은 기능을 제공합니다:
    • AI 기반 블로그 콘텐츠 생성
    • SEO(검색엔진최적화) 분석
    • 콘텐츠 유사도 검사
    • SERP(검색결과) 미리보기
    • 업종별 템플릿 제공
  2. 서비스는 브라우저 기반으로 제공되며, 별도의 회원가입 없이 이용 가능합니다.
  3. AI 기능 사용을 위해서는 이용자가 직접 발급받은 API 키가 필요합니다.

제5조 (이용자의 의무)

이용자는 다음 행위를 하여서는 안 됩니다:

  1. 타인의 API 키를 무단으로 사용하는 행위
  2. 서비스를 이용하여 불법적인 콘텐츠를 생성하는 행위
  3. 서비스의 정상적인 운영을 방해하는 행위
  4. 타인의 명예를 훼손하거나 권리를 침해하는 콘텐츠를 생성하는 행위
  5. 저작권, 상표권 등 지적재산권을 침해하는 행위
  6. 허위 정보를 유포하는 콘텐츠를 생성하는 행위
  7. 서비스를 역설계, 디컴파일 또는 해킹하려는 행위
  8. 서비스를 이용하여 스팸 또는 대량 자동 생성 콘텐츠를 만드는 행위

제6조 (콘텐츠의 저작권)

  1. 서비스를 통해 생성된 콘텐츠의 저작권은 이용자에게 귀속됩니다.
  2. 이용자는 생성된 콘텐츠에 대한 모든 책임을 부담합니다.
  3. AI가 생성한 콘텐츠는 기존 저작물과 유사할 수 있으므로, 이용자는 게시 전 독창성을 확인할 책임이 있습니다.

제7조 (API 키 관리)

  1. 이용자는 자신의 API 키를 안전하게 관리해야 합니다.
  2. API 키 사용으로 인해 발생하는 비용은 전적으로 이용자의 책임입니다.
  3. 서비스는 이용자의 API 키를 수집하거나 저장하지 않습니다(브라우저 localStorage 제외).
  4. 공용 컴퓨터에서 사용 시 브라우저 데이터 삭제를 권장합니다.

제8조 (서비스 중단)

  1. 서비스는 다음 경우에 서비스 제공을 일시 중단할 수 있습니다:
    • 시스템 점검, 보수 또는 교체가 필요한 경우
    • 천재지변, 전쟁, 폭동 등 불가항력적 사유가 발생한 경우
    • 외부 API 서비스(OpenAI, Google, Anthropic)의 장애가 발생한 경우
  2. 서비스 중단으로 인한 이용자의 손해에 대해 서비스는 책임을 지지 않습니다.

제9조 (면책조항)

  1. 서비스는 다음 사항에 대해 책임을 지지 않습니다:
    • 이용자가 생성한 콘텐츠의 정확성, 적법성, 신뢰성
    • AI가 생성한 콘텐츠로 인해 발생하는 법적 분쟁
    • 이용자의 API 키 유출로 인한 피해
    • 외부 AI 서비스의 장애 또는 변경으로 인한 서비스 이용 불가
    • 이용자의 귀책사유로 인한 서비스 이용 장애
  2. 서비스는 이용자가 서비스를 통해 얻은 정보 또는 자료로 인해 발생한 손해에 대해 책임을 지지 않습니다.
  3. AI가 생성한 콘텐츠는 참고용이며, 최종 게시 전 이용자가 검토해야 합니다.

제10조 (손해배상)

  1. 이용자가 본 약관을 위반하여 서비스 또는 제3자에게 손해를 끼친 경우, 이용자는 그 손해를 배상할 책임이 있습니다.
  2. 서비스의 책임 있는 사유로 이용자에게 손해가 발생한 경우, 서비스는 관련 법령에 따라 배상할 수 있습니다.

제11조 (분쟁해결)

  1. 서비스와 이용자 간에 발생한 분쟁에 관한 소송은 대한민국 법원을 관할 법원으로 합니다.
  2. 서비스와 이용자 간에 발생한 분쟁에 대해서는 대한민국 법령을 적용합니다.

제12조 (기타)

  1. 본 약관에서 정하지 않은 사항은 관련 법령 및 상관례에 따릅니다.
  2. 본 약관의 일부 조항이 무효가 되더라도 나머지 조항은 유효합니다.
