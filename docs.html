빠른 시작

AI 블로그 도구는 회원가입 없이 바로 사용할 수 있습니다. 아래 3단계만 따라하면 됩니다.

1

API 키 준비

OpenAI, Google Gemini, 또는 Anthropic Claude 중 하나의 API 키를 준비하세요. Google Gemini는 무료 티어가 있어 비용 없이 시작할 수 있습니다.

2

주제 입력

작성할 블로그 글의 주제, 키워드, 원하는 톤앤매너를 입력하세요. 더 자세히 입력할수록 좋은 결과물이 나옵니다.

3

생성 및 편집

'초안 생성' 버튼을 클릭하면 AI가 글을 작성합니다. 생성된 초안을 검토하고 필요에 따라 편집하세요.

API 키 설정

AI 블로그 도구는 사용자의 API 키를 사용합니다. 각 AI 서비스에서 API 키를 발급받는 방법을 안내합니다.

🔑 OpenAI API

  1. OpenAI 플랫폼에 가입합니다.
  2. API Keys 페이지로 이동합니다.
  3. "Create new secret key" 버튼을 클릭합니다.
  4. 생성된 키를 복사하여 안전한 곳에 저장합니다.
  5. AI 블로그 도구의 "API 설정"에 붙여넣기 합니다.

💡 GPT-4o와 GPT-4o-mini 모델을 사용할 수 있습니다. GPT-4o-mini가 더 저렴합니다.

🌟 Google Gemini API

  1. Google AI Studio에 Google 계정으로 로그인합니다.
  2. 왼쪽 메뉴에서 "Get API key"를 클릭합니다.
  3. "Create API key" 버튼을 클릭합니다.
  4. 프로젝트를 선택하고 키를 생성합니다.
  5. 생성된 키를 AI 블로그 도구에 입력합니다.

💡 무료 티어로 시작할 수 있어 비용 부담 없이 테스트해볼 수 있습니다.

🤖 Anthropic Claude API

  1. Anthropic Console에 가입합니다.
  2. 대시보드에서 "API Keys" 섹션으로 이동합니다.
  3. "Create Key" 버튼을 클릭합니다.
  4. 키 이름을 입력하고 생성합니다.
  5. 생성된 키를 AI 블로그 도구에 입력합니다.

💡 Claude 3.5 Sonnet은 빠르고 효율적이며, Claude 3 Opus는 더 정교한 글을 생성합니다.

⚠️ 보안 주의사항

API 키는 비밀번호처럼 안전하게 관리해야 합니다. 공용 컴퓨터에서 사용 후에는 반드시 브라우저 데이터를 삭제하세요.

첫 글 작성하기

처음 사용하신다면 아래 단계를 따라 첫 블로그 글을 작성해보세요.

1단계: 설정 확인

  • API 키가 정상적으로 입력되었는지 확인합니다.
  • 사용할 AI 모델을 선택합니다 (처음이라면 기본값 추천).

2단계: 주제 입력

  • 주제: 작성할 글의 핵심 주제를 입력합니다. (예: "홈 카페 에스프레소 머신 추천")
  • 키워드: SEO를 위한 핵심 키워드를 입력합니다. (예: "에스프레소 머신, 홈카페, 가정용 커피머신")
  • 톤앤매너: 글의 분위기를 선택합니다. (예: "친근하고 정보전달적인")
  • 추가 지시: 특별히 포함하고 싶은 내용이 있다면 입력합니다.

3단계: 생성 및 편집

  • "초안 생성" 버튼을 클릭하고 기다립니다 (보통 10-30초).
  • 생성된 초안을 꼼꼼히 읽어봅니다.
  • 필요에 따라 내용을 수정하거나 추가합니다.
  • SEO 분석 탭에서 최적화 점수를 확인합니다.
  • 유사도 검사로 표절 위험을 체크합니다.

글 구조 선택

사전 정의된 구조를 선택하거나 직접 구조를 지정할 수 있습니다:

  • 기본 구조: 서론 → 본론 → 결론
  • 리스트형: 순위나 목록 형태
  • 문제-해결형: 문제 제기 → 해결책 제시
  • 비교 분석형: 여러 대상 비교
  • 직접 입력: 원하는 구조 직접 작성

SEO 분석

작성한 글이 검색엔진에 얼마나 최적화되어 있는지 분석합니다.

분석 항목

제목 분석

제목 길이, 키워드 포함 여부, 매력도 평가

메타 설명

적정 길이, 키워드 배치, 클릭 유도 문구

키워드 밀도

본문 내 키워드 분포 및 적정 비율

가독성

문장 길이, 단락 구성, 헤딩 구조

콘텐츠 품질

글 길이, 정보의 깊이, 독창성

구조 분석

H1-H6 태그 사용, 목록, 이미지 alt

SEO 점수 해석

  • 90-100점: 매우 우수 - 검색 상위 노출에 유리
  • 70-89점: 양호 - 약간의 개선으로 더 좋아질 수 있음
  • 50-69점: 보통 - 주요 개선 사항 적용 권장
  • 50점 미만: 개선 필요 - SEO 요소 전반적 검토 필요

유사도 검사

작성한 글이 기존 웹 콘텐츠와 얼마나 유사한지 검사합니다.

검사 방식

로컬 검사 (기본)

브라우저 내에서 텍스트 패턴을 분석합니다. API 비용이 들지 않지만, 실제 웹 콘텐츠와의 비교는 제한적입니다.

AI 기반 검사

AI를 활용하여 더 정밀한 유사도 분석을 수행합니다. 문맥적 유사성까지 감지할 수 있습니다.

결과 해석

  • 0-20%: 독창적인 콘텐츠
  • 21-40%: 약간의 유사성 - 일반적인 표현이 포함됨
  • 41-60%: 주의 필요 - 일부 문장 재작성 권장
  • 61% 이상: 높은 유사도 - 상당 부분 재작성 필요

템플릿 활용

미리 만들어진 템플릿을 사용하면 더 빠르고 효과적으로 글을 작성할 수 있습니다.

템플릿 사용 방법

  1. 템플릿 라이브러리에서 원하는 템플릿을 찾습니다.
  2. "템플릿 사용하기" 버튼을 클릭합니다.
  3. 도구 페이지로 이동하면 해당 템플릿 구조가 자동 적용됩니다.
  4. 주제와 키워드만 입력하면 해당 구조로 글이 생성됩니다.

제공 템플릿 종류

  • 마케팅: 제품 리뷰, 서비스 비교, 브랜드 스토리
  • IT/테크: 기술 튜토리얼, 도구 가이드, 트렌드 분석
  • 건강: 건강 정보, 레시피, 운동 루틴
  • 교육: 개념 설명, 학습 로드맵, 시험 가이드
  • 비즈니스: 산업 분석, 케이스 스터디, 인터뷰
  • 라이프스타일: 여행, 인테리어, 취미 가이드

자주 묻는 질문

API 키 발급 자체는 무료입니다. 다만 API 사용량에 따라 비용이 청구됩니다. Google Gemini는 무료 티어가 있어 일정량까지 무료로 사용할 수 있습니다.

AI가 생성한 콘텐츠의 저작권은 사용자에게 있습니다. 다만 AI 생성 콘텐츠의 저작권에 대한 법적 해석은 국가마다 다를 수 있으므로, 상업적 사용 시 관련 법률을 확인하시기 바랍니다.

API 키는 사용자의 브라우저 localStorage에만 저장됩니다. 우리 서버로 전송되거나 저장되지 않습니다. 브라우저에서 AI API로 직접 통신합니다.

먼저 API 키가 정확히 입력되었는지, 해당 API 서비스의 크레딧이 남아있는지 확인하세요. 네트워크 연결 상태도 점검해주세요. 문제가 지속되면 다른 AI 모델로 변경해보세요.

네, 모바일 브라우저에서도 사용 가능합니다. 다만 긴 글 편집 시 PC 환경이 더 편리할 수 있습니다.

네, 도구 페이지에서 이전에 생성한 글 목록을 확인할 수 있습니다. 히스토리는 브라우저 localStorage에 저장되므로, 같은 브라우저에서만 확인 가능합니다.

네, AI 모델들은 다국어를 지원합니다. 추가 지시 란에 원하는 언어를 명시하면 해당 언어로 글이 생성됩니다. (예: "영어로 작성해주세요")

현재는 한 번에 하나의 글만 생성됩니다. 첫 번째 글 생성이 완료된 후 다음 글을 생성하시면 됩니다.

생성된 글은 마크다운 형식으로 복사하거나 직접 선택하여 복사할 수 있습니다. 에디터에서 Ctrl+A로 전체 선택 후 복사하면 됩니다.

현재 모든 기능을 무료로 제공하고 있으며, 향후 프리미엄 기능이 추가되더라도 현재의 무료 기능은 그대로 유지할 예정입니다.

문제 해결

문제가 발생했을 때 시도해볼 수 있는 해결 방법입니다.

🔴 "API 키가 유효하지 않습니다" 오류

  • API 키가 정확히 복사되었는지 확인하세요.
  • 키 앞뒤에 공백이 없는지 확인하세요.
  • 해당 API 서비스의 콘솔에서 키가 활성화되어 있는지 확인하세요.
  • 새 API 키를 발급받아 시도해보세요.

🔴 글 생성이 너무 느립니다

  • 요청한 글 길이가 너무 길지 않은지 확인하세요.
  • 더 빠른 모델(GPT-4o-mini, Gemini Flash)로 변경해보세요.
  • 인터넷 연결 상태를 확인하세요.
  • AI 서비스 제공업체의 서버 상태를 확인하세요.

🔴 생성된 글의 품질이 낮습니다

  • 입력 정보(주제, 키워드)를 더 구체적으로 작성하세요.
  • 추가 지시에 원하는 내용을 자세히 설명하세요.
  • 더 고급 모델(GPT-4o, Claude Opus)을 사용해보세요.
  • 적절한 템플릿을 선택하여 구조를 잡아주세요.

🔴 히스토리가 사라졌습니다

  • 브라우저의 캐시나 localStorage를 삭제하면 히스토리도 삭제됩니다.
  • 다른 브라우저나 시크릿 모드에서는 히스토리가 공유되지 않습니다.
  • 중요한 글은 별도로 복사하여 저장해두시기 바랍니다.

🔴 SEO 점수가 낮게 나옵니다

  • 제목에 핵심 키워드가 포함되어 있는지 확인하세요.
  • 메타 설명을 155자 내외로 작성하세요.
  • 본문에 H2, H3 등 소제목을 적절히 사용하세요.
  • 글 길이가 너무 짧지 않은지 확인하세요 (최소 300단어 권장).

위 방법으로 해결되지 않는 문제가 있다면 연락처로 문의해주세요.
