개인정보처리방침

시행일: 2026년 1월 1일 | 최종 수정: 2026년 1월 27일

1. 개요

AI 블로그 도구(이하 "서비스")는 사용자의 개인정보를 중요시하며, 「개인정보 보호법」 및 관련 법령을 준수하고 있습니다. 본 개인정보처리방침은 서비스가 수집하는 정보의 종류, 사용 목적, 보관 방법 등을 설명합니다.

2. 수집하는 정보

본 서비스는 서버를 통한 개인정보 수집을 하지 않습니다. 모든 데이터는 사용자의 브라우저(localStorage)에만 저장됩니다.

2.1 브라우저에 저장되는 정보

  • API 키: 사용자가 입력한 OpenAI, Google Gemini, Anthropic Claude API 키
  • 작성 내용: 블로그 주제, 키워드, 생성된 콘텐츠
  • 설정 정보: 사용자가 선택한 AI 모델, 글자 수 등의 설정
  • 히스토리: 이전에 생성한 블로그 초안 기록

2.2 서버로 전송되지 않는 정보

위에 나열된 모든 정보는 사용자의 브라우저에만 저장되며, 당사 서버로 전송되거나 수집되지 않습니다.

3. 제3자 서비스

본 서비스는 사용자가 직접 입력한 API 키를 사용하여 다음 서비스와 통신합니다:

3.1 AI API 서비스

  • OpenAI API: 블로그 콘텐츠 생성에 사용
    OpenAI 개인정보처리방침
  • Google Gemini API: 블로그 콘텐츠 생성에 사용
    Google 개인정보처리방침
  • Anthropic Claude API: 블로그 콘텐츠 생성에 사용
    Anthropic 개인정보처리방침

3.2 광고 서비스

  • Google AdSense: 광고 표시를 위해 쿠키를 사용할 수 있습니다.
    Google 광고 정책

3.3 폰트 서비스

  • Google Fonts: 웹 폰트 제공
    Google Fonts 개인정보 FAQ

4. 쿠키 사용

본 서비스는 자체적으로 쿠키를 사용하지 않습니다. 다만, 다음의 제3자 서비스에서 쿠키를 사용할 수 있습니다:

  • Google AdSense: 맞춤 광고 제공을 위한 쿠키
  • Google Fonts: 폰트 캐싱을 위한 쿠키

사용자는 브라우저 설정을 통해 쿠키를 차단하거나 삭제할 수 있습니다.

5. 데이터 보관 및 삭제

5.1 보관 기간

브라우저 localStorage에 저장된 데이터는 사용자가 직접 삭제하거나 브라우저 데이터를 초기화할 때까지 보관됩니다.

5.2 데이터 삭제 방법

  • 서비스 내 "설정" 메뉴에서 "탭 닫으면 자동 삭제" 옵션 활성화
  • 브라우저 개발자 도구 > Application > Local Storage에서 직접 삭제
  • 브라우저 설정에서 사이트 데이터 삭제

6. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

  • 저장된 데이터에 대한 접근 권리
  • 저장된 데이터의 삭제 권리
  • 서비스 사용 중단 권리

모든 데이터가 사용자의 브라우저에 저장되므로, 사용자가 직접 데이터를 관리할 수 있습니다.

7. 아동의 개인정보

본 서비스는 만 14세 미만의 아동을 대상으로 하지 않습니다. 만 14세 미만의 아동이 서비스를 이용하려면 법정대리인의 동의가 필요합니다.

8. 보안

본 서비스는 다음과 같은 방식으로 사용자 데이터를 보호합니다:

  • HTTPS를 통한 암호화된 통신
  • API 키의 브라우저 내 로컬 저장 (서버 미전송)
  • 중간 서버 없이 AI API와 직접 통신

9. 개인정보처리방침의 변경

본 개인정보처리방침은 관련 법령 변경이나 서비스 변경에 따라 수정될 수 있습니다. 변경 시 본 페이지에 공지하며, 중요한 변경의 경우 서비스 내 알림을 통해 안내합니다.
